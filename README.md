# 右键菜单管理器

一个用于 **Windows** 的右键菜单管理工具（基于 Python + Tkinter）。它会读取系统注册表中的右键菜单项，展示来源程序、创建时间和注册表路径，并支持搜索、查看详情与删除。

## 功能
- 扫描并展示右键菜单项（支持文件/目录相关项）
- 实时搜索过滤
- 查看详细信息（来源程序、创建时间、注册表路径）
- 右键快捷操作：打开注册表位置、复制注册表路径
- 删除选中的菜单项（需要管理员权限）

## 运行环境
- Windows 10/11
- Python 3.8+（建议使用官方 Python）
- 依赖：`pywin32`、`pillow`

## 安装依赖
```bash
pip install -r requirements.txt
```

## 运行
```bash
python context_menu_manager.py
```

首次运行会请求管理员权限（用于访问和删除注册表项）。

## 生成快捷方式
项目内提供了生成快捷方式脚本：
```bash
python create_shortcut.py
```
运行后会在当前目录生成：
- `右键菜单管理器.lnk`
- `启动右键菜单管理器.bat`

## 使用说明
- 双击列表项可查看详细信息。
- 右键列表项可快速打开注册表位置或复制注册表路径。
- 删除操作会直接删除注册表项，请谨慎使用。

## 注意事项
- 删除注册表项可能影响系统或软件功能，请提前确认并自行备份。
- 本工具仅操作 `HKEY_CLASSES_ROOT` 下的 `*\shell` 和 `Directory\shell`。
